# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

Web design agency workspace. Projects involve cloning/recreating existing websites, then customizing design and content for clients. Full stack — frontend UI through backend APIs and data.

## Project Conventions

- Each client project lives in its own subdirectory (e.g., `projects/client-name/`)
- Reference/source sites are documented so the original design intent is clear
- Customizations (colors, fonts, copy, images) are separated from structural/layout code where possible
